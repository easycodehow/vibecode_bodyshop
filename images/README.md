## 사용 방법

CSS에서 이미지를 사용하려면:
```css
.slide1 .slide-bg {
    background-image: url('../images/slide1.jpg');
}
```

HTML에서 이미지를 사용하려면:
```html
<img src="images/category-face.jpg" alt="페이스 케어">
```

## 주의사항

- 이미지는 웹 최적화된 형식 사용 (JPG, PNG, WebP)
- 파일 크기는 가능한 작게 유지 (500KB 이하 권장)
- 반응형을 위해 여러 해상도 준비 고려
